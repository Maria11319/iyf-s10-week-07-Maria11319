## Project Description

A project focused on JavaScript best practices including local storage, state management, clean code principles, debugging, and code organization. Includes a Shopping Cart app, Notes app, and various daily challenges.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- localStorage API
- ESLint
- Prettier
- npm

## Features

- Shopping Cart with state management
- Notes app with localStorage persistence
- Dark mode theme toggle
- Recent search history
- Form auto-save
- Centralized state management pattern
- Observer pattern implementation
- ESLint and Prettier setup

## How to Run

1. Clone this repository
2. Open any HTML file in your browser:
    - `cart.html` for Shopping Cart
    - `notes.html` for Notes App
    - `theme.html` for Theme Toggle
    - `search.html` for Recent Searches
    - `autosave.html` for Form Auto-Save
